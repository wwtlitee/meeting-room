# Meeting Room 成品需求预设

Use this file when the user describes the deliverable they want rather than naming a specialist. These presets prevent duplicate persona creation by routing overlapping requests to the best existing role plus optional collaborators.

## 调度规则

- 用户说“做游戏”时，先选 Browser Game Producer，再按需要加入 Web Game Engineer、Game UI HUD Designer、Game QA Playtester。
- 用户说“做视频”时，先选 Short Video Producer，再按需要加入 Storyboard Director、Voiceover Scriptwriter、Subtitle Packaging Specialist、FFmpeg Automation Engineer。
- 用户说“做工具/脚本”时，先选 Utility Product Designer，再按技术形态加入 CLI Tool Engineer、Python Automation Script Engineer、PowerShell Automation Specialist、Desktop Utility Engineer。
- 用户说“做网页/UI”时，先根据页面类型选 Landing Page Architect、Admin Dashboard Designer、Website Information Architect 或 SaaS Dashboard Product Designer，再加入 Responsive Layout Specialist 和 Visual QA Inspector。
- 用户说“做文案”时，先按平台/用途选 Brand Copywriter、Sales Page Copywriter、Short Video Hook Writer、Email Sequence Copywriter、Community Announcement Writer。
- 用户说“做数据/表格/看板”时，先选 Spreadsheet Analyst 或 Business Insight Analyst，再按交付物加入 Data Visualization Designer、Dashboard Builder、Chart Storytelling Editor。
