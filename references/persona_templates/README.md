# Persona Templates

本目录保存 `references/personas/<division>/<slug>/` 的初始化模板。模板只描述人格资料层，不替代 `references/roles/` 下的源角色卡。

## 文件

- `PROFILE_TEMPLATE.md`：稳定身份与职责边界模板 - v1.0.0
- `KNOWLEDGE_TEMPLATE.md`：长期专业知识库模板 - v1.0.0
- `MEMORY_TEMPLATE.md`：近期原始记忆模板 - v1.0.0
- `MEMORY_SUMMARY_TEMPLATE.md`：压缩摘要与策略模板 - v1.0.0

## 占位符

脚本会替换以下占位符：

- `{{name}}`
- `{{slug}}`
- `{{division}}`
- `{{description}}`
- `{{role_path}}`
- `{{generated_at}}`
