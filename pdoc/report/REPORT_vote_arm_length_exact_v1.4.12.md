# REPORT_vote_arm_length_exact_v1.4.12

## 变更时间

2026-05-28

## 背景

SVG 弯臂第一版将 `length` 当作手腕位置，末端圆掌额外伸出，导致视觉长度超过用户在设置中调好的手臂长度；闭合块造型也偏厚重。

## 变更内容

- 将 `--vote-arm-length` 改为 SVG 手臂视觉总宽度，不再额外叠加圆掌半径。
- 将闭合块轮廓改为圆角曲线路径加小圆掌，降低厚度与弯曲幅度。
- 保留原肩膀锚点、初始手位、投票手位、按钮位置与长度设置。

## 复核

- 已执行 `npm run build`，构建通过。
- 逻辑复核重点：设置里的手臂长度现在直接约束视觉总长；按钮反馈点仍锚在 `--vote-arm-length`。
