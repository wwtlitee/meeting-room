# REPORT_vote_arm_taper_and_contact_cleanup_v1.4.13

## 变更时间

2026-05-28

## 背景

投票手臂按下后会出现额外红/绿色触碰块，和按钮亮灯反馈重复；同时 SVG 曲线手臂为等宽 stroke，缺少肩粗腕细的剪影感。

## 变更内容

- 移除 `stage-vote-arm-contact` 触碰光点，避免手部下方出现红/绿色块。
- 将手臂从等宽 stroke 改为填充式闭合剪影。
- 新轮廓保持 `length` 为视觉总长，肩部加粗、腕部收窄、手端保留小圆弧。
- SVG 关闭外溢显示，避免收回状态下肩部向身体外侧倒冒。

## 复核

- 已执行 `npm run build`，构建通过。
- 逻辑复核重点：按钮亮灯仍由按钮自身动画负责；手臂不再引入额外 A/B 色块。
