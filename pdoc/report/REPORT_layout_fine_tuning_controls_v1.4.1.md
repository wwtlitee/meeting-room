# REPORT_layout_fine_tuning_controls_v1.4.1

## 基本信息

- 日期：2026-05-28
- 版本：v1.4.1
- 负责人：Solazhu
- 范围：专家团 viewer 设置面板布局精修、铭牌角度、投票按钮尺寸与全座位校准

## 修改内容

- 设置面板新增 `精修` 区，配合拖动做精确调整。
- 铭牌目标支持：
  - `X / Y` 位置微调；
  - `旋转` 平面角度；
  - `朝向` 透视 `rotateY`；
  - `俯仰` 透视 `rotateX`；
  - `斜切`；
  - `宽`。
- 投票按钮目标支持：
  - `X / Y` 位置微调；
  - `旋转`；
  - `大小` 缩放。
- 铭牌 CSS 接入 `--plate-yaw` 和 `--plate-pitch`，可把左/右铭牌调整为朝向桌面中心的透视席卡。
- 投票按钮 CSS 接入 `--vote-buttons-scale`，可在设置中调整整体大小。
- 校准目标改为全座位槽位：即使当前会议某个座位为空，也会在设置里出现对应的铭牌和按钮目标，便于一次性调完布局。
- `SKILL.md` 增加规则：用于视觉/布局调试的重新拉起会议应尽量补满所有可见座位，避免空位无法校准。

## 影响范围

- 影响 React viewer 设置面板与舞台元素样式。
- 不改变会议 JSON 必填字段。
- 旧的布局 localStorage 仍兼容；新增字段只在用户微调后写入。

## 复核结果

- `npm run build` 通过。
- 当前内置浏览器已确认设置面板出现 `精修`。
- 铭牌目标已确认出现 `朝向`、`俯仰`、`斜切`、`宽`。
- 全座位校准已确认出现空位槽位 `right4`、`left5`、`right5`。
