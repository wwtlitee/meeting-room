## 修改内容

- 大屏主题压缩逻辑从“截断冒号前文本”改为“提炼产品名 + 会议类型”，例如 `deadman 记忆对话 app 优化方向...` 显示为 `deadman 记忆对话优化会`。
- 主题提炼会剔除 `app/application`、引号和冗余空格，并把 `优化方向/优化讨论会/项目优化讨论会` 统一收束为 `优化会`。
- 大屏标题字体从像素字体切换为更紧凑的中文 UI 字体栈，降低英文字符占位，避免正常短标题也被 CSS 省略。

## 风险与防护

- 仅影响大屏幕展示标题，不改变会议数据、历史记录标题或 Codex 输出消息。
- 未识别到 `记忆对话` 的其他会议仍保留通用截断逻辑，避免过度改写陌生项目名。
- CSS 仍保留 `text-overflow: ellipsis` 兜底，防止异常超长主题压住 A/B 投票区域。
