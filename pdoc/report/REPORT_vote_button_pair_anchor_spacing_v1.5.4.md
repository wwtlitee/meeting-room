# REPORT_vote_button_pair_anchor_spacing_v1.5.4

## 背景

用户要求十个席位的投票按钮组合都以靠下面的按钮为锚点，上面的按钮向下收一点，让两枚按钮更靠近。

## 修改内容

- 将 `.seat-vote-buttons` 从普通 `inline-flex + gap` 改为固定 `50px x 15px` 的局部坐标系。
- 左侧席位保持下方 A 按钮原位，将上方 B 按钮从 `left: 28px` 收到 `left: 23px`。
- 右侧席位保持下方 B 按钮原位，将上方 A 按钮从 `left: 0` 收到 `left: 5px`。
- 保持按钮组整体坐标、旋转、缩放不变，避免破坏用户已校准的十个位置。

## 复核

- 已执行 `npm run build`，Vite 构建通过。
- 需要人工确认：左右两侧每组按钮的下方按钮是否仍在原锚点，上方按钮是否更靠近。
