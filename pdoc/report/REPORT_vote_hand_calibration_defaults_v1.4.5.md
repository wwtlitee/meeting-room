# REPORT_伸手动作校准默认值_v1.4.5

## 变更摘要

- 从 Codex 内置浏览器 `Local Storage/leveldb` 中读取 `agency-agents-nameplate-calibration-v2` 的最新记录，提取用户手工调整后的 10 个座位伸手起点。
- 新增 `voteHandPlacements` 默认起点表，清空浏览器 localStorage 后仍会保留本次校准结果；伸手终点继续跟随投票按钮中心，避免按钮旋转影响下压方向。
- 设置面板将 `测试 A / 测试 B` 改为 `伸手测试 A / 伸手测试 B`，用于直接观察伸手与亮灯效果。
