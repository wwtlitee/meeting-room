# REPORT_投票手臂视觉修正_v1.4.8

## 变更摘要

- 将 `stage-vote-arm` 的主体从尖头裁剪改为圆头胶囊形态，避免手部末端像尖刺。
- 投票手臂层级改为跟随席位 `z - 1`，在渲染顺序上位于对应员工身体图层下方。
- 调整 `vote-arm-rotate` 动画节奏，明确分成伸出、短暂停留、收回三段，收回段比之前更长、更容易被看见。

## 影响范围

- `App.jsx`：为每条手臂注入 `--vote-arm-z`。
- `styles.css`：调整手臂层级、形态和伸出/收回关键帧。

## 复核

- 已执行 `npm run build`，Vite 构建通过。
- 坐标数据未改动；保留用户手动校准后的布局。
