# Expert Meeting Character Sprite Prompts

Version: v1.0.0

These prompts are for later character layers. Do not generate these until the empty meeting room base is approved and the anchor points are measured.

## Prompt A: Cow Expert Neutral Sprite

```text
Use case: background-extraction
Asset type: 2D character sprite for web animation
Primary request: Generate a cute modular cartoon cow expert character sitting in a meeting chair, front-facing with a slight inward turn, suitable for overlaying on a conference-room background.
Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for background removal.
Subject: one small cartoon cow expert, black and white body, rounded head, small horns, simple friendly face, seated posture, visible upper body and head, simple colored chest badge area that can be recolored later. No full office environment.
Style/medium: clean 2D mascot illustration, soft 3D-like shading, Marvis-inspired black silhouette readability but cuter and original, crisp edges for animation.
Composition/framing: centered sprite with generous padding, no crop, suitable for 128x128 to 256x256 display.
Lighting/mood: soft studio lighting, no cast shadow on the background.
Color palette: black, white, light gray, one removable accent color on the badge.
Text (verbatim): no text.
Constraints: one character only, no background elements, no logo, no watermark, no text. Background must be one uniform #00ff00 color with no gradients, no floor, no shadows, no texture. Do not use #00ff00 anywhere in the character.
Avoid: realistic cow, horror style, complex fur, many accessories, tiny unreadable details, extra characters.
```

## Prompt B: Speaking Mouth Frames

Use this to create a simple sprite sheet after the neutral character direction is approved.

```text
Use case: background-extraction
Asset type: 2D character sprite sheet for web animation
Primary request: Generate a sprite sheet of the same modular cartoon cow expert character with four speaking mouth frames.
Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for background removal.
Subject: four evenly spaced frames in one row, same cow expert head and upper body, identical pose and size in all frames. Only the mouth changes: closed, small open, wide open, smiling closed. Include a simple colored chest badge area that can be recolored later.
Style/medium: clean 2D mascot illustration, soft 3D-like shading, crisp edges, original Marvis-inspired office mascot mood.
Composition/framing: 4-frame horizontal sprite sheet, each frame equal size, consistent baseline, generous padding.
Lighting/mood: soft studio lighting, no cast shadow.
Color palette: black, white, light gray, one removable accent badge color.
Text (verbatim): no text.
Constraints: exactly four frames, one cow character per frame, no text, no logo, no watermark. Background must be one uniform #00ff00 color. Do not use #00ff00 in the character.
Avoid: changing body pose between frames, changing camera angle, extra accessories, extra characters, uneven frame sizes.
```

## Prompt C: Role Accessory Sheet

Use this when role identities need visual variety without generating a full new mascot per expert.

```text
Use case: background-extraction
Asset type: accessory sprite sheet for modular character customization
Primary request: Generate a clean accessory sprite sheet for a cartoon cow expert character.
Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for background removal.
Subject: small separate accessories only: headset, glasses, small cap, tiny crown-like moderator marker, leaf pin, star pin, code badge, design pen badge, chart badge, shield badge. Each accessory should be isolated, front-facing, simple, and readable at small UI size.
Style/medium: clean 2D UI mascot accessories, soft shading, crisp edges.
Composition/framing: grid layout, equal spacing, generous padding around each accessory.
Lighting/mood: soft studio lighting, no shadows touching the background.
Color palette: black, white, light gray, muted blue, teal, yellow, purple accents.
Text (verbatim): no text.
Constraints: accessories only, no full characters, no logo, no watermark, no readable words. Background must be one uniform #00ff00 color.
Avoid: complex detailed icons, realistic props, text labels, extra background elements.
```
