# PROMPT_five_body_sprite_asset_board_v1.4.0

Create a production asset board for an expert-meeting viewer. The goal is not a finished scene screenshot; it is a modular source sheet for later sprite and furniture extraction.

## Direction Lock

- Use the reference direction: strong black cow silhouettes, white office chairs and table, small colored seat badges, clean bright meeting-room styling.
- Keep exactly five reusable base body types: standard, round, tall, small, sturdy.
- Persona differences may only use tiny accessories, external nameplates, small badges, and role props.
- Do not create one unique body model per expert persona.

## Required Sections

1. Empty bright meeting room background, no table, no chairs, no characters, no text.
2. Vertical long white conference table plus separate left, right, and bottom occluder strips.
3. White chair directions: left back/front, right back/front, host back/front.
4. Five cow seed bodies: standard, round, tall, small, sturdy. Each must use a readable black silhouette and consistent bottom-center anchor.
5. Action planning strips for walk-left, walk-right, walk-up, idle, talk, nod, challenge, think. These can be schematic if the seed bodies are the primary focus.
6. Accessory and external nameplate area: glasses, headset, pen, cap, chart, shield, notebook, tiny neutral badges.

## Style Keywords

modular 2D game asset board, bright white office furniture, black cow expert silhouettes, clean production sheet, high oblique orthographic view, transparent-ready components, crisp edges, small muted accent badges, readable sprite anchors

## Negative Prompt

finished meeting screenshot, all characters baked into one scene, low-contrast gray-white cow bodies, high saturation body fills, large colored torso bars, one unique model for every expert, text baked into sprites, noisy background, heavy shadows, cropped assets, inconsistent scale, independent frame generation drift
